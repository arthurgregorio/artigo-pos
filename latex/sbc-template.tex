\section{Introdução}

De acordo com \cite{xavier2013} arquitetura \'e a definição dos elementos que comp\~oem uma estrutura e como estes v\~ao se relacionar, assim podemos dizer que a cria\c{c}\~o de uma arquitetura de software \'e algo que exige do arquiteto uma s\'erie de conhecimentos espec\'ificos sobre como estes componentes funcionam ou como podem interagir entre s\'i.

Definir uma arquitetura não \'e algo simples, pois mais importante do que saber integrar os compomentes selecionados, \'e saber identificar quais as poss\'iveis escolhas e quais os fatores que influenciam nestas escolhas \cite{xavier2013}.
